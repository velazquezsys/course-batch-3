Web components

  1. Crear un componente que contenga un botón toggle.
  2. Crear otro componente que contenga una lista con 
                    <ul>
                    <li></li>
                    ...
                <ul>
  3. Los elementos <li> deben ser recibidos a través de un atributo/propiedad del componente 2. Tiene que ser un array de strings, donde cada entrada es un elemento de la lista.
  4. El primer componente (botón toggle) deberá mostrar/ocultar el segundo componente (lista).
  • Basarse en los conceptos de template, shadowDom, html import y custom elements. (Deben incluir los 4 conceptos). Los ejemplos están en el repo alfonsorios96/webComponents
  • Usar binding visto en clase, y customEvents para comunicar los dos componentes.
  • Cada webComponent deberá tener su propio archivo html y deberán ser importados en un tercer documento HTML llamado index.html.
  • Subirlo a su repositorio antes de las 22h00 del día 21 de junio de 2018, UTM -6:00 Ciudad de México
